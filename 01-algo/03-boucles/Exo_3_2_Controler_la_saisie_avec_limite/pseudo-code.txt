L’utilisateur est invité à saisir un mot de passe.

Si le mot de passe saisi est correct, le programme affiche “Vous êtes connecté”.

Dans le cas contraire, l’utilisateur doit recommencer la saisie.

L’utilisateur dispose de 3 essais maximum.

Au 3ème échec, le programme affiche “Votre compte est bloqué” et se termine.

Note : Le bon mot de passe est formation

VARIABLES

	password est une chaine de caracteres
	
	compteur est un entier

START PROGRAM

PROCESSING

	compteur <-- 1

		DO 
			
			WRITE "Tapez votre mot de passe!"

	        READ password
		
		START IF 

		if password n'est pas formation and  le compteur est égal à 1
			
			WRITE "Mot de passe incorrect, vous devez réessayer ; vous avez encore 2 tentatives"

		else if password n'est pas formation and  le compteur est égal à 2
			
			WRITE "Mot de passe incorrect, vous devez réessayer ; vous avez encore 1 tentative"

		else 

			WRITE "Votre compte est bloqué"

		END IF

		WHILE password est "formation" and le compteur est inférieur ou égal à 3

			WRITE "Vous êtes connecté"


END PROCESSING

END PROGRAM
